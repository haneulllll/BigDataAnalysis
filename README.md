# 🚢 AIS 빅데이터 처리 및 해상 네트워크 분석

이 저장소는 **AIS(Automatic Identification System) 데이터 처리 및 해상 네트워크 분석**을 위한 프로젝트입니다.  
대규모 AIS 데이터를 효율적으로 처리하고, 머신러닝을 활용하여 선박 경로를 예측하며, 대시보드를 통해 실시간으로 데이터를 시각화하는 기능을 포함합니다.
